--- layout: default title: Snippets - garmin-auto-import - Import data from your Garmin device ---

Description

garmin-auto-import runs in the background and detects when you connect your Garmin device. It will then synchronize the data into your Documents/garmin folder, and prompt you for more actions to do.

Screenshots

Synopsis

Help:
$ garmin-auto-import --help

Run it
$ garmin-auto-import

Upload files manually:
$ garmin-auto-import upload Documents/garmin/2011-09-04T18:08:09Z.tcx ...

Install

# garmin-auto-import
sudo wget http://github.com/jhthorsen/snippets/blob/master/bin/garmin-auto-import \
    -O /usr/local/bin/garmin-auto-import
chmod +x /usr/local/bin/garmin-auto-import

# garmin-sync
wget http://launchpad.net/garmin-sync/devel/0.3/+download/garmin-sync-0.3.tar.gz
tar xfzv garmin-sync-0.3.tar.gz
sudo cp garmin-sync-0.3/* /usr/local/bin/
rm -r garmin-sync-0.3*

# Mojolicious
wget -q http://cpanmin.us | perl - --sudo Mojolicious

# The rest...
sudo apt-get install \
    getopt-long-descriptive \
    libdevice-usb-perl \
    libipc-run3-perl \
    python-usb

Run it

Requirements

garmin-sync, Perl, Device::USB, Getopt::Long::Descriptive, IPC-Run3, Mojolicious and Gnome environment (nautilus, send-to, gnome-open, zenity),
